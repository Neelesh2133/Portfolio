## 🚀 Featured Projects Highlighted
The portfolio highlights three key end-to-end AI/ML systems I have designed:

### 1. **EduTailor** — LLM & ML Learning Path Generator
* **Description**: A personalized learning recommendation platform utilizing stacking ensemble classifiers (Random Forest, Gradient Boosting, XGBoost) and ChromaDB vector search with Sentence Transformers to optimize LLM pathing.
* **Impact**: Achieved **92% prediction accuracy** and reduced average response latency from **2.5s to 1.3s**.
* **Stack**: Python, FastAPI, React, XGBoost, ChromaDB, Llama-3, Gemini, Docker, DVC, MLflow.
* **Repository**: [EduTailor Repo](https://github.com/Neelesh2133/Edutailor)

### 2. **UrDoc** — Deep Learning X-Ray Classifier
* **Description**: A full-stack medical helper web application classifying bone X-rays using transfer learning with a fine-tuned MobileNetV2. Features robust API validation and Grad-CAM activation visualization.
* **Impact**: Delivers **98.4% true positive rate** with model predictions served in under **1 second**.
* **Stack**: Python, PyTorch, MobileNetV2, FastAPI, Next.js, React, Tailwind CSS, Firebase.
* **Repository**: [UrDoc Repo](https://github.com/toufiq13/UrDoc)

### 3. **CodeSage** — AI-Powered Code Review Tool
* **Description**: A developer tool analyzing codebases and GitHub pull requests using NVIDIA Nemotron 3 Ultra LLM. Implements custom chunking, severity scoring, and structured JSON generation.
* **Stack**: Python, FastAPI, PostgreSQL, React, Docker, NVIDIA Nemotron.
* **Repository**: [CodeSage Repo](https://github.com/Neelesh2133/CodeSage1)

## 🛠️ Getting Started Locally
To run this portfolio locally on your machine, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Neelesh2133/Portfolio.git
   cd Portfolio
   ```

2. **Open index.html**:
   * Simply open the `index.html` file in any modern web browser.
   * Alternatively, run a local development server for the best experience (e.g. using Live Server in VS Code, or Python):
     ```bash
     python -m http.server 8000
     ```
     Then open `http://localhost:8000` in your browser.
